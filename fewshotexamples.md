## 예시 1: 주행거리 보정 — 반복 수렴 방식

### 상황
- **대상차량**: 모델X, 트림A, 5년차, 15만km, 출고가 3000만원
- **DB 검색 결과**: 같은 모델+트림+연식, 키로수만 다른 차량 3대

| 기준차 | 주행거리 | 낙찰가 |
|--------|---------|--------|
| A | 18만km | 1,500만원 |
| B | 14만km | 1,700만원 |
| C | 20만km | 1,350만원 |

### 추론 과정

**1차 보정 (공식 적용: 만km당 60만원 = 출고가 3000 × 2%)**

- 기준차A (18만km → 15만km): 1500 + 60×3 = **1,680만원**
- 기준차B (14만km → 15만km): 1700 - 60×1 = **1,640만원**
- 갭: 1,680 - 1,640 = **40만원** → 공식이 정확하지 않음

**2차 보정 (감가율 조정: 60만 → 55만/만km)**

- 기준차A: 1500 + 55×3 = **1,665만원**
- 기준차B: 1700 - 55×1 = **1,645만원**
- 갭: 1,665 - 1,645 = **20만원** → 줄어듦

**3차 보정 (감가율 조정: 55만 → 52만/만km)**

- 기준차A: 1500 + 52×3 = **1,656만원**
- 기준차B: 1700 - 52×1 = **1,648만원**
- 갭: 1,656 - 1,648 = **8만원** → 거의 수렴

**검증 (기준차C로 확인)**

- 기준차C (20만km → 15만km): 1350 + 52×5 = **1,610만원**
- 약간 낮음 → 고키로 구간에서 감가 둔화 가능성 (20만km 천장 효과)
- 15만km 구간에서는 기준차A, B 기반 **약 1,650만원**이 적정

### 결론
- 이 차종은 만km당 약 52만원 감가 (공식 60만원보다 낮음)
- 15만km 대상차량 예상 낙찰가: **약 1,650만원**

## 예시 2: 이상치 조사 — 가격이 안 맞는 차 원인 추론

### 상황
- **대상차량**: 모델Y, 트림B, 4년차, 8만km
- **DB 검색 결과**: 같은 모델+트림+연식, 비슷한 키로수 차량 4대

| 기준차 | 주행거리 | 낙찰가 | 비고 |
|--------|---------|--------|------|
| A | 7만km | 2,100만원 | |
| B | 9만km | 2,050만원 | |
| C | 8만km | 2,350만원 | ← 혼자 높음 |
| D | 8만km | 1,800만원 | ← 혼자 낮음 |

### 추론 과정

**1단계: 정상 범위 확인**
- A, B를 8만km로 보정하면 약 2,050~2,080만원 → 이게 정상 범위
- C는 +300만원 높고, D는 -250만원 낮음 → 왜?

**2단계: 이상치 C 조사 (get_vehicle_detail 호출)**
- 옵션 확인 → 선루프, 스마트키, 네비, 후방카메라 **풀옵션**
- 풀옵션 보정: 50만×4 = 200만원 → 2,350 - 200 = 2,150만원
- 나머지 +100만원은 색상 프리미엄(흰색) 가능성

**3단계: 이상치 D 조사**
- 확인 → 수출 차량 (내수가 아님)
- 또는 사고이력(교환 3개+) 확인
- → 기준차에서 제외

### 결론
- 기준차 A, B 기반 → 대상차량 예상 낙찰가: **약 2,070만원**
- 기준차 C는 풀옵션+색상 프리미엄으로 설명 가능 → 옵션 보정 후 사용 가능
- 기준차 D는 수출 or 사고차 → 기준차에서 제외

## 예시 3: 출고가 비율 활용 — 희귀 차종 빠른 추론

### 상황
- **대상차량**: 희귀 모델Z, 11만km, 출고가 4000만원
- **DB 검색 결과**: 같은 모델 데이터가 2건뿐

| 기준차 | 주행거리 | 낙찰가 | 출고가 | 낙찰/출고 비율 |
|--------|---------|--------|--------|--------------|
| A | 10만km | 1,500만원 | 3,000만원 | 50% |
| B | 11만km | 1,440만원 | 3,000만원 | 48% |

### 추론 과정

**1단계: 비율 패턴 파악**
- 10만km → 50%, 11만km → 48%
- 만km당 약 2% 하락 → 공식과 부합

**2단계: 대상차량에 적용**
- 대상차량: 출고가 4,000만원, 11만km
- 11만km → 48% 적용
- 4,000 × 0.48 = **1,920만원**

### 결론
- 데이터 2건뿐이지만, 출고가 비율이 일관적 (만km당 2%)
- 대상차량 예상 낙찰가: **약 1,920만원**
- 신뢰도: 보통 (데이터 2건, 출고가 다른 차량에 비율 적용)

## 핵심 추론 원칙 요약

1. **기본 세팅**: 모델+트림+연식 고정 → 키로수나 출고가(≈옵션) 비슷한 것 찾기
2. **공식 먼저, 조정은 나중에**: 업계 감가 공식으로 1차 보정 → 여러 기준차 비교하며 갭 확인 → 감가율을 조정하여 수렴
3. **이상치는 버리지 말고 조사**: 가격이 안 맞으면 원인(수출/옵션/사고/색상) 찾기 → 설명 가능하면 보정 후 활용, 아니면 제외
4. **출고가 비율**: 출고가가 있으면 낙찰/출고 비율(%)로 빠르게 추론 가능. 특히 희귀 차종에 유용
5. **교차검증**: 최소 2~3대로 추정하고, 결과가 수렴하면 신뢰할 수 있음
